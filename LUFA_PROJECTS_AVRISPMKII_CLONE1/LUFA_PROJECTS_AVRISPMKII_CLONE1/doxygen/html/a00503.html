Detailed Description

USB class driver for the USB-IF MIDI class standard.

Module Source Dependencies

The following files must be built with any user project that uses this module:

  • LUFA/Drivers/USB/Class/Device/MIDIClassDevice.c (Makefile source module name: LUFA_SRC_USBCLASS)
  • LUFA/Drivers/USB/Class/Host/MIDIClassHost.c (Makefile source module name: LUFA_SRC_USBCLASS)

Module Description

MIDI Class Driver module. This module contains an internal implementation of the USB MIDI Class, for both Device and Host USB modes. User applications can use this class driver instead of implementing the MIDI class manually via the low-level LUFA APIs.

This module is designed to simplify the user code by exposing only the required interface needed to interface with Hosts or Devices using the USB MIDI Class.

Note
The USB MIDI class is actually a special case of the regular Audio class, thus this module depends on structure definitions from the Audio 1.0 Class Device Mode Driver class driver module.
